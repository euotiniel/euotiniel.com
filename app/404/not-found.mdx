
## 404

“404” é um código usado pela web para indicar que uma página não foi encontrada.

Normalmente isso significa uma de três coisas:

- o link está errado;
- a página foi removida;
- ou o site mudou enquanto este endereço ficou para trás.

De qualquer forma, provavelmente vais querer voltar ao início.
